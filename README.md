# ofxImageHomography
4-point image homography test for openframework

![Histogram example]( https://github.com/bemoregt/ofxHistogramFeatures/blob/master/hist.png "exmaple")

### How TO
- Move your mouse to warp.

### Dependency
- OpenFrameworks 0.10.1
- ofxCv
- ofxOpenCv
- ofxHitogram
- XCode 10.12.x
- OSX Mojave
